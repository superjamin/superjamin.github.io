java集合-Set

发表于 2020-08-10 | 分类于 java基础

Set是java中的集合对象，用来存放一组无序且不重复的数据。

Set主要实现类有HashSet、LinkedHashSet、TreeSet。

下面通过例子来具体分析它们：

可以看到重复的数据没被放入set，打印出来的顺序也与放置进去的顺序不同，小结以下几点：

1、HashSet是无序的

2、允许插入null

3、不支持存放重复数据

4.是线程不安全的

二、LinkedHashSet

总结以下LinkedHashSet：

1、是有序的，放入取出和顺序相同

2、支持插入null

3、不能插入重复数据

4、是非线程安全的

5、使用链表维护插入顺序

三、TreeSet

可以看到在插入null时报错了，我们知道TreeSet时SortedSet的唯一实现，通过Comparator接口来对对象进行排序，插入null报错因为null没实现Comparator接口，我们来写个例子验证下

输出结果：

可以看到null插入成功，因为TreeSet在执行add（）方法时，会调对象的comparate（）方法进行比较，放入TreeSet的对象需要实现Comparator接口，Java中的常用类比如String、Character等已经实现了Comparable接口’

总结一下TreeSet：

1、不允许插入重复数据

2、使用红黑树实现，数据处于排序状态

3、添加元素时会比较，所以最好是同一类对象

所以我们可以自己定义Comparator，传入TreeSet的构造函数

ps：Comparable和Comparator区别

  • 一个类实现了 Comparable 接口，意味着该类的对象可以直接进行比较（排序），但比较（排序）的方式只有一种，很单一。
  • 一个类如果想要保持原样，又需要进行不同方式的比较（排序），就可以定制比较器（实现 Comparator 接口）。
